هيكل HTML


يستخدم قالب HTML أحدث Bootstrap v5.0 مع علامات HTML5 صالحة.هذا الموضوع هو تخطيط تفاعلي مع 4 اعمدة دعم .جميع المعلومات في منطقة المحتوى متداخل داخل Class ويأتي مع Class محددة مسبقا.

للتعديل يجب اولا عليك بتثبيت برنامج Visual Studio Code ثم فتح المشروع علي البرنامج عن طريق الدخول علي ملفات الموقع والضغط علي الزر الايمن للماوس واختيار open with code ثم البحث عن السكشن المناسب الذي تريد تعديله من ثم التعديل والحفظ وسيظهر معك التحديث بشكل تلقائي في المتصفح ويمكنك البحث عن طريق "ctrl + f" ثم كتابة النص الذي تبحث عنه وسيظهر لك ثم التعديل وللحفظ نضغط علي "ctrl + s"

ملفات CSS والهيكل


يتم استخدام ملف وحيد CSS وهو الرئيسي بشكل أساسي في هذا الموضوع style.css

إذا كنت ترغب في تحرير قسم معين من الموقع ، فما عليك سوى العثور على التسمية المناسبة في ملف CSS ، ثم التمرير لأسفل حتى تجد النمط المناسب الذي يجب تحريره. ويمكنك البحث عن طريق "ctrl + f" ثم كتابة class الذي تبحث عنه وسيظهر لك ثم التعديل وللحفظ نضغط علي "ctrl + s"
